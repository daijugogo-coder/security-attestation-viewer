# Security Attestation

## Scope
This document describes the security checks performed for the PDF to PowerPoint
conversion web application.

## Performed Checks
- Static analysis using Bandit
- Dependency vulnerability check using pip-audit
- Cloud Run configuration review (environment variables, secret handling)

## Not Performed
- Third-party security audit
- Penetration testing
- Formal security certification
